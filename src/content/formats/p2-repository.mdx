# P2 Repository

Cloudsmith provides public & private repositories for Eclipse/Equinox P2

P2 provides general-purpose infrastructure provisioning for software applications, including installing Eclipse and Equinox-based applications.

For more information on P2, please see:

- [P2](https://www.eclipse.org/equinox/p2/)
- [P2 Wiki](https://wiki.eclipse.org/Equinox/p2)

<Note variant="warning" headline="Early Access Availability">
P2 is available via feature flag. Please contact support.
</Note>
