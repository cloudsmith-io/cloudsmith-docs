# Package Managers

## List of Package Managers

|Assets|Package Format|
|:-- |:-- |
|[Alpine Linux](https://alpinelinux.org/) based packages. For example: Distributions based on Alpine, such as popular Docker images.|[Alpine](alpine-repository)|
|[Swift](https://developer.apple.com/swift/) / [Objective-C](https://developer.apple.com/library/archive/documentation/Cocoa/Conceptual/ProgrammingWithObjectiveC/Introduction/Introduction.html) packages.|[Swift](swift-registry)[Cocoapods](cocoapods-repository)|
|[PHP Composer](https://getcomposer.org/) project/library (.phar) packages.|[Composer](composer-repository) (Packagist-compatible)|
|[C, C++](https://conan.io/) packages.|[Conan](conan-repository)|
|[Powershell](https://learn.microsoft.com/en-us/powershell/) and [Chocolatey](https://chocolatey.org/) packages are used for system administration and automation on windows systems|[PowerShell Modules Repository](powershell-modules-repository)  [Chocolatey Repository](chocolatey-repository)|
|[Rust](https://www.rust-lang.org/) packages.|[Cargo](cargo-registry) (Crates.io-compatible)|
|[Dart](https://dart.dev/)/[Flutter](https://flutter.dev) packages.|[Dart](dart-repository)|
|[Debian Linux](https://www.debian.org/) (.deb) based packages.  Example distributions: Debian, Ubuntu, Elementary, Raspian, Mint, etc.|[Debian](debian-repository) (deb)|
|[Go](https://github.com/golang/go/wiki/Modules) packages (modules).|[Go](go-registry)|
|[Docker](https://docker.com) images.|[Docker](docker-registry)|
|[Hex](https://hex.pm/) packages for the BEAM ecosystem; any language that compiles to run on the BEAM VM, such as elixir and Erlang, can be used to build Hex packages.|[Hex](hex-repository)|
|[Kubernetes](https://kubernetes.io/) Helm charts.|[Helm](helm-chart-repository)|
|[Lua](https://www.lua.org/) modules (called rocks - it's a whole moon theme!).|[LuaRocks](luarocks-repository)|
|[Maven-based](https://maven.apache.org/) or compatible packages, for Java, Scala, Clojure, Kotlin, or other similar languages.  Also supports [Gradle](https://gradle.org), Sbt, Leiningen and other build systems.|[Maven](maven-repository)|
|[npm](https://docs.npmjs.com/about-npm/) project/library packages.|[npm](npm-registry) (npmjs-compatible)|
|[NuGet](https://www.nuget.org) (.nupkg, .snupkg, etc.) packages.|[NuGet](nuget-feed)|
|[Python](https://www.python.org/) project/library (.whl, .zip, .tar.gz, etc.) packages.|[Python](python-repository) (PyPi-compatible)|
|[R](https://www.r-project.org) statistical computing and graphics packages.|[CRAN](cran-repository)|
|[Raw / Generic](https://en.wikipedia.org/wiki/File_server) / (any type) packages, which is basically _anything_ that you want to version and distribute.  For example: Windows binaries, Go packages, Datasets, etc.|[Raw](raw-repository) / Generic|
|[Ruby](https://www.ruby-lang.org/en/) project/library (.gem) packages.|[Ruby](ruby-repository)|
|[RedHat Linux](https://www.redhat.com/en/technologies/linux-platforms/enterprise-linux) based packages,  Example distributions: RedHat, CentOS, SUSE, Oracle, Scientific, etc.|[RedHat](redhat-repository) (rpm)|
|[Terraform Module](https://www.terraform.io/docs/modules/index.html) packages|[Terraform](terraform-modules-repository)|
|[Unity Package Manager](https://docs.unity3d.com/Manual/Packages.html)|[Unity Registry](unity-registry)|
|[Vagrant](https://www.vagrantup.com/) boxes.|[Vagrant](vagrant-repository)
